## Get Started Here

Welcome to the Artifaq documentation! Artifaq is a Python web framework built on top of FastAPI, designed to streamline project organization and enhance your development experience. Explore Artifaq's key features below and start building high-performance, modular applications.

<CardGrid stagger>
	<Card title="FastAPI Performance" icon="rocket">
		Harness the speed and efficiency of FastAPI to build fast, scalable APIs with Artifaq.
	</Card>
	<Card title="Optimized Project Structure" icon="seti:folder">
		Artifaq enforces a rigid project organization to keep you productive right from the start.
	</Card>
	<Card title="Simplified Configuration" icon="pen">
		Enjoy automatic configuration loading for a hassle-free project setup.
	</Card>
	<Card title="Modularity" icon="puzzle">
		Organize your applications into reusable, independent modules for better maintainability.
	</Card>
	<Card title="Automatic Routing" icon="magnifier">
		Artifaq automatically discovers and registers your routes, making route management simple.
	</Card>
	<Card title="Built-in Security" icon="forward-slash">
		Includes basic implementations for authentication and authorization to secure your application.
	</Card>
	<Card title="Flexible ORM Integration" icon="seti:puppet">
		Easily integrate with various ORMs like SQLAlchemy or SQLModel to manage your databases flexibly.
	</Card>
</CardGrid>
